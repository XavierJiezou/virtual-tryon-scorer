# 🎯 Virtual Try-On Scorer（虚拟试穿效果打分）

[English](./README.en.md) | **简体中文**

一个用于评估 AI 虚拟试穿（VTON）效果质量的 Agent Skill。通过分析身份保持、服装还原、身体一致性和背景稳定性四个维度，对试穿结果进行结构化评分。

## ✨ 功能特点

- 🧑 **人脸身份保持评估** — 检查试穿前后人脸是否一致，这是最关键的指标
- 👕 **服装还原与贴合度评估** — 检查目标服装的颜色、图案、版型是否还原，穿着是否自然
- 🦵 **非人脸身体特征保持评估** — 检查体型、肤色、手部、配饰等是否保持一致
- 🏞️ **背景保持评估** — 检查场景、光照、边缘融合是否稳定
- 📊 **加权总分** — 根据各维度重要性聚合为一个总分

## 📐 评分体系

| 维度 | 权重 | 核心检查项 |
|------|------|------------|
| 人脸身份保持 | **40%** | 面部结构、五官、肤色、表情、发型 |
| 服装还原与贴合 | **30%** | 颜色、图案、版型、面料质感、贴合度 |
| 非人脸身体特征保持 | **20%** | 体型比例、肤色、手部、纹身/痣、配饰 |
| 背景保持 | **10%** | 场景一致性、光照、伪影、边缘融合 |

**总分 = 人脸 × 0.4 + 服装 × 0.3 + 身体 × 0.2 + 背景 × 0.1**

每项评分范围为 **0–100 分**，附有简短的文字说明。

## 📥 输入格式

支持两种输入方式：

### 方式一：三张独立图片
分别上传：
1. **原始人物照片**（试穿前的人物图）
2. **目标服装图片**（要试穿的衣服参考图）
3. **试穿结果图片**（AI 生成的虚拟试穿效果图）

### 方式二：单张拼接图
上传一张包含上述三张照片拼接在一起的图片（横向排列、纵向排列或网格排列均可），Skill 会自动识别各个面板的内容。

## 📤 输出示例

```
## 虚拟试穿效果评分报告

### 输入识别
- 输入格式：三张独立图片
- 原始人物：亚洲女性，长发，穿白色T恤
- 目标服装：红色碎花连衣裙，V领设计
- 试穿结果：同一女性穿着红色碎花连衣裙

### 分项评分

#### 1. 人脸身份保持 (权重 40%)
- **得分：92/100**
- 评价：面部五官高度一致，肤色自然，仅下巴轮廓略有柔化。

#### 2. 服装还原与贴合 (权重 30%)
- **得分：85/100**
- 评价：红色碎花图案还原良好，V领结构准确，但左袖口处花纹略有变形。

#### 3. 非人脸身体特征保持 (权重 20%)
- **得分：88/100**
- 评价：体型比例一致，手部自然，手腕手表保留完整。

#### 4. 背景保持 (权重 10%)
- **得分：95/100**
- 评价：室内背景完全一致，光照自然，人物边缘过渡平滑。

### 总分
- **加权总分：89.9/100**
- 计算方式：(92 × 0.4) + (85 × 0.3) + (88 × 0.2) + (95 × 0.1)

### 总体评价
整体试穿效果优秀，人脸身份保持出色，服装还原度高。
主要改进方向为左袖口处的花纹细节还原。
```

## 🔧 安装方式

### 全局安装（所有项目可用）

将 `virtual-tryon-scorer` 文件夹复制到全局 skills 目录：

**Antigravity IDE:**
```bash
cp -r virtual-tryon-scorer ~/.gemini/antigravity/skills/
```

**Claude Code:**
```bash
cp -r virtual-tryon-scorer ~/.claude/skills/
```

**通用 Agent（.agent 标准）:**
```bash
cp -r virtual-tryon-scorer ~/.agent/skills/
```

### 项目级安装（仅当前项目可用）

```bash
mkdir -p .agents/skills
cp -r virtual-tryon-scorer .agents/skills/
```

## 🚀 使用方式

安装后，在对话中直接提出试穿评分相关的需求即可自动触发，例如：

- "帮我给这个虚拟试穿结果打个分"
- "评价一下这张试穿效果图的质量"
- "这个 VTON 结果怎么样？"
- 上传试穿前后的照片并说"对比一下试穿效果"
